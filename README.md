# Movie Recommendation System (Python + NumPy)

This project implements a **Movie Recommendation System** using only **Python + NumPy**.  

---

## ASCII Diagram of Pipeline
User Ratings --> User-Item Matrix --> Collaborative Filtering --> Recommended Movies


---

## Features
- Collaborative Filtering implemented from scratch with NumPy
- ASCII-based visualizations
- Works with simple Python lists / NumPy arrays
- Can scale to more users and movies
